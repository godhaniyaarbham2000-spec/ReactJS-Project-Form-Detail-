# 📝 ReactJS Registration Form Project

A modern **React + Vite Registration Form** with full form validation, error handling, and success submission display.  
Built to understand controlled components, validation logic, and React state management.

---

## 🚀 Features

* 👤 Full Name, Email, Password, City, Phone form fields  
* ⚡ Real-time form validation  
* 🚫 Error messages for invalid inputs  
* 🧠 Controlled components using React state  
* ⏳ Simulated form submission delay  
* ✅ Success message with submitted data preview  
* 🎯 Clean UI with validation feedback  

---

## 🛠️ Tech Stack

* React.js (useState, useEffect)
* JavaScript (ES6+)
* CSS3 (Custom styling, Flexbox)
* HTML5 (JSX structure)
* Vite (Build tool)

---

## 📌 Topics Used

* React Functional Components  
* useState Hook  
* useEffect Hook  
* Form Handling in React  
* Controlled Components  
* Form Validation Logic  
* Conditional Rendering  
* JavaScript Regex Validation  
* Event Handling (onChange, onSubmit)

## ⚙️ How to Run This Project

### 1. Clone the repository
```
git clone https://github.com/godhaniyaarbham2000-spec/ReactJS-Project-Form-Validation.git
```



### 2. Move to project folder
```
cd ReactJS-Project-Form-Validation
```


### 3. Install dependencies
```
npm install
```



### 4. Start development server
```
npm run dev
```

---

## 💡 Usage

- Fill all input fields (Name, Email, Password, City, Phone)  
- Errors will appear if input is invalid  
- Submit button enables only when form is valid  
- On submit, success message + data is shown  
- Data resets after submission
